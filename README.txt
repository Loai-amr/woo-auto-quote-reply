Automatically sends professional WooCommerce-styled emails in response to YITH Request a Quote submissions.

== Description ==

Woo Auto Quote Reply extends WooCommerce and YITH Request a Quote functionality by sending customers automatic, professional email responses after they submit a quote request.  

The email includes:
* Product names and details  
* Quantities requested  
* Individual pricing and line totals  
* Overall quote total  
* WooCommerce email styling  

Also includes debug and test tools for admins.

== Features ==

* Automatic customer response emails
* Supports multiple YITH hook versions
* WooCommerce integration with pricing + email templates
* Admin dashboard with testing + debugging tools
* Debug logging for troubleshooting
* Built-in test email system

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woo-auto-quote-reply/` directory, or install the plugin through the WordPress Plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Ensure WooCommerce and YITH Request a Quote (Free or Pro) are active.
4. Optionally, test the plugin using the built-in test email feature in the admin dashboard.
